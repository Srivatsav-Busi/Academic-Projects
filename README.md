## üöÄ Getting Started

### Prerequisites

- Python 3.8 or higher
- Jupyter Notebook or JupyterLab
- Git
- (Optional) Docker for containerized projects

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Srivatsav-Busi/Academic-Projects.git
   cd Academic-Projects
   ```

2. **Set up Python environment** (recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies** (project-specific):
   ```bash
   # For specific projects, navigate to project directory
   cd "Project Name"
   pip install -r requirements.txt
   ```

### Running Projects

#### Jupyter Notebook Projects
```bash
jupyter notebook "Project Name/notebook.ipynb"
```

#### Python Applications
```bash
# Example: CustomGPT
cd CustomGPT
python run.py
```

#### Web Applications
